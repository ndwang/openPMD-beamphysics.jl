I/O

File Format

This package reads and writes files in the openPMD-beamphysics HDF5 format.

Reading Particles

using BeamPhysics

# Load particles (file must contain exactly one particle path)
pg = ParticleGroup("particles.h5")

# Inspect available paths in a file
using HDF5
h5open("particles.h5", "r") do h5
    println(particle_paths(h5))
end

See ParticleGroup, particle_paths.

Writing Particles

write_particle_group("output.h5", pg)

See write_particle_group.

Reading Field Meshes

fm = FieldMesh("fieldmesh.h5")

# Inspect available paths
h5open("fieldmesh.h5", "r") do h5
    println(field_paths(h5))
end

See FieldMesh, field_paths.

Writing Field Meshes

write_fieldmesh("output.h5", fm)

See write_fieldmesh.

Low-Level HDF5 API

For advanced use, the following functions are also exported. See the API Reference for full signatures.

Reading: particle_paths, field_paths, load_field_attrs, is_constant_component, constant_component_value, component_data, offset_component_name, particle_array, all_components, component_str

Writing: pmd_init, pmd_field_init, write_component_data, write_pmd_bunch, write_pmd_field, particle_data_dict, write_particle_group
